### 문제 설명

<p>포스텍 학생회관에는 포닉스가 좋아하는 미끄럼틀이 있다.</p>

<p style="text-align: center;"><img alt="" src="https://upload.acmicpc.net/8ba42534-cefe-4a98-818b-cb3066b8993c/-/preview/"></p>

<p style="text-align: center;"><small>포닉스가 좋아하는 미끄럼틀</small></p>

<p>포스텍에는 포닉스가 좋아하는 장소가 총 $N$곳 있다. 포닉스는 2차원에 살기 때문에 포닉스가 좋아하는 장소는 아래 조건을 만족하는 2차원 좌표평면 위의 점으로 표현할 수 있다.</p>

<ul>
	<li>모든 장소의 $x$좌표와 $y$좌표는 $1$ 이상 $N$ 이하의 정수이다.</li>
	<li>모든 장소의 $x$좌표는 서로 다르다.</li>
	<li>모든 장소의 $y$좌표는 서로 다르다.</li>
</ul>

<p>즉, $N$개 장소의 위치는 모든 $1\le i\le N$에 대해 좌표 $(i,A_i)$로 표현된다.</p>

<p>학생회관의 미끄럼틀이 인상 깊었던 포닉스는 $N$개의 장소를 한 번에 오갈 수 있도록 모든 장소 쌍마다 미끄럼틀을 설치했다. 두 장소 $(i,A_i)$ , $(j,A_j)$를 골라 만든 미끄럼틀의 경사를 $\vert\frac{A_j-A_i}{j-i}\vert$로 정의한다.</p>

<p>포닉스는 용감하기 때문에 가장 경사가 큰 미끄럼틀을 탈 것이다. 반면, 찬우는 미끄럼틀을 무서워하기 때문에 가장 경사가 작은 미끄럼틀을 탈 것이다.</p>

<p>각 장소의 좌표가 주어질 때, 포닉스와 찬우가 탈 미끄럼틀의 경사를 각각 구하여라.</p>

### 입력 

 <p>첫 번째 줄에 장소의 수 $N$이 주어진다. $(2\leq N\leq 300 000)$</p>

<p>두 번째 줄에 각 장소의 $y$좌표를 나타내는 $N$개의 정수 $A_1,\cdots ,A_N$이 공백으로 구분되어 주어진다. $(1\le A_i\le N)$</p>

<p>모든 $A_i$는 서로 다름이 보장된다.</p>

### 출력 

 <p>첫 번째 줄에 포닉스가 탈 미끄럼틀의 경사를 기약분수 형태로 출력한다.</p>

<p>두 번째 줄에 찬우가 탈 미끄럼틀의 경사를 기약분수 형태로 출력한다.</p>
