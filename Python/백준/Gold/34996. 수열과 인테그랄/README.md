<p>토카의 망상으로 만들어진 인테그랄 양이 진심 공부를 전력으로 방해하고 있다!</p>

<p>불쌍하게도 토카는 다른 과목의 시험이 얼마 남지 않아서 시험 준비에 몰두해야한다. 토카는 인테그랄 양의 제안을 정중히 거절하려 했지만 인테그랄 양은 내기를 제안했다.</p>

<p>"네가 정 그렇다면... 내가 내는 문제를 맞춘다면 <strong>작은 선물</strong>을 줄게!"</p>

<p>선물이라니! 토카는 선물이라면 사족을 쓰지 못하기 때문에 당연히 인테그랄 양의 제안을 받아들였다.</p>

<p>인테그랄 양은 다음과 같은 문제를 냈다.</p>

<div style="background:#eeeeee;border:1px solid #cccccc;padding:5px 10px;"><strong>문제. </strong>절댓값이 $N$ 이하인 서로 다른 $2N+1$개의 정수로 이루어진 수열 $A$에 대해, 정의역이 $[1,2N+1]$인 함수 $f$를 $x$가 정수일 때 $f(x) = A_x$, 그 외의 경우에는 $i = \lfloor x \rfloor$라 할 때 $f(x) = A_i + (A_{i+1} - A_i)\,(x - i)$와 같이 정의하자. 함수 $f$의 그래프와 $x$축 사이 넓이가 최대가 되게 하는 수열 $A$와 그 때의 넓이를 구하시오. 함수 $f$의 그래프와 $x$축 사이의 넓이는 $\int_{1}^{2N+1} \bigl|f(x)\bigr| \,dx$로 정의한다.</div>

<p>당신은 토카 대신 문제를 풀어 토카가 선물을 받을 수 있도록 도와주려 한다. 토카를 대신해 인테그랄 양의 문제를 해결해보자.</p>

### 입력 

 <p>첫 번째 줄에 정수 $N$ $(1 \le N \le 5\,000)$이 주어진다.</p>

### 출력 

 <p>첫 번째 줄에 $f$의 그래프와 $x$축 사이 넓이의 최대값을 출력한다. 절대 또는 상대 오차는 ${10}^{-4}$까지 허용한다.</p>

<p>두 번째 줄에 $f$의 그래프와 $x$축 사이 넓이가 최대가 되게 하는 수열 $A$를 공백으로 구분하여 한 줄에 출력한다. 수열은 $-N$부터 $N$까지의 서로 다른 정수로 이루어져야 한다.</p>

<p>가능한 수열이 여러 개라면 아무거나 출력한다.</p>
